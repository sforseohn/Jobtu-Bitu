## 🤝 스터디 규칙

### 1️⃣ 장소와 시간

→ 매주 금요일 8시 비대면으로 대략 2시간 동안 진행

### 2️⃣벌금

→ **주 당 최소 문제 개수:** 주제 3문제 (고정) + 구현 1문제 (각자 난이도 조정)

→ **벌금:** 못 푼 문제 개수 x 1000원

단, 계절 기간 동안 못 푼 구현 문제를 방학 내에 다 풀면 인정

### 3️⃣ 진행 방식

1.  매주 비대면으로 만나 각자의 풀이를 공유한다
2.  해당 알고리즘에 대해 이해가 안가는 부분은 서로 이야기하며 해결한다.
